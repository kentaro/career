About あんちぽっぷ@エロ壁紙こみゅん

趣意書

エロ！ ああ、なんて甘美な響き。僕らは毎日々々エロいことばかり考えて暮らしているのではなかったか。そして僕は毎日々々 Web site をどうすれば面白くしていけるかを考えている。となれば、毎日そのことばかり考えているエロと Web site がやがては結合して組んずほぐれつ、ねっとりべっちょりとまさぐりあっているうちに、どちらの穴が入れてもいい穴なのかわからなくなってくるのは必然…。

…ごめんなさい。エロいことなんて全然書いてないです。

事務的な記述

当サイトは全ドキュメントを XHTML 1.1 にて記述し、デザインを CSS にて行っております。完全に strict な記述であるとは決していえませんが、できるだけ適切な記述を目指しています。ちなみに、outsider reflex 提供のスタイルシート切替スクリプトを悪用して、ちょっとした仕掛けが施してあります。詳細については、"StyleSelect" という表記のそばに配されたリスト・メニューをぐりぐりしてみてください。

About あんちぽっぷ：あやや SIDE

趣意書

当サイトは誰がなんといおうとも松浦亜弥さんのファン・サイトです。熱烈ラヴです。そんなことは御覧になれば誰にでもわかることです。しかし、あややのことを僕はほとんど知らないので、全く関係ない愚痴みたいなことばっかり書いてます。でも、あややを好きな気持ちでは誰にも負けません。実はあややとつき合ってるし。

…まぁ、そんな感じで。

事務的な記述

当サイトは全ドキュメントを XHTML 1.1 にて記述し、デザインを CSS にて行っております。完全に strict な記述であるとは決していえませんが、できるだけ適切な記述を目指しています。ちなみに、outsider reflex 提供のスタイルシート切替スクリプトを悪用して、ちょっとした仕掛けが施してあります。詳細については、"StyleSelect" という表記のそばに配されたリスト・メニューをぐりぐりしてみてください。デフォルトのスタイルがエロい方に設定されていますが、あやや SIDE を選択された場合、次回お越しの際はあややスタイルが適用されたデザインで表示されるはずです。エロい方に戻りたいときは、スタイル選択メニューを再度ぐりぐりしてください。
